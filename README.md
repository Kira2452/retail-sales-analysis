# Retail Sales Data Analysis

## Описание проекта
Пет-проект по анализу данных о розничных продажах.  
В проекте выполнен исследовательский анализ данных (EDA), рассчитаны ключевые метрики,
построены визуализации и проверена гипотеза с помощью A/B теста.

## Используемые инструменты
- Python
- pandas, numpy
- matplotlib, seaborn
- scipy
- Jupyter Notebook

## Структура проекта
- `retail_sales_analysis.ipynb` — основной ноутбук с анализом  
- `retail_sales_dataset.csv` — датасет с данными  

## Основные шаги анализа
1. Загрузка и очистка данных  
2. Исследовательский анализ данных  
3. Расчёт бизнес-метрик  
4. Визуализация данных  
5. Проверка гипотезы (A/B тест)

## Гипотеза
Средний чек отличается между мужчинами и женщинами.

## Результат
Получен результат статистического теста, показывающий наличие / отсутствие
статистически значимой разницы в среднем чеке между группами.
